Visão Geral:

Este projeto é um bot de automação de tarefas desenvolvido em python, utilizando as bibliotecas PyAutoGUI e Pandas, para preencher um cadastro de produtos com dados provenientes de uma base de dados.

Funcionalidades Principais:

Automação: O bot simula ações do usuario, como clicks de mouse e teclado, etc.
Manipulação de Dados: A biblioteca Pandas é usada para ler e processar os dados da base, garantindo que os produtos sejam cadastrados corretamente.

Como funciona:

  1. Assim que iniciado o bot abre o navegador e realiza o login na plataforma.
  2. Ele lê a base de dados (nesse caso, um arquivo CVS).
  3. Para cada produto na base, o bot preenche os campos necessários na página.
  4. O processo de cadastro de produtos é repetido até que todos os produtos sejam cadastrados.

Benefícios:

  • Eficência: O bot automatiza tarefas repetitivas, economizando tempo.
  • Escabilidade: Pode lidar com grandes quantidades de dados de forma consistente
  • Precisão: Os dados são inseridos exatamente conforme especificado na base de dados
